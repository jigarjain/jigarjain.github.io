# Website

Source code for my personal site. Please visit [jigarjain.com](https://jigarjain.com) instead


## Development

- Install RVM
```sh
curl -L https://get.rvm.io | bash -s stable
```
- Install Ruby(2.6) via RVM
```sh
rvm --default use 2.6.4
```
- Install bundle
```sh
gem install bundle
```
- Install all gems from Gemfile
```sh
bundle install
```
- Serve the site
```sh
jekyll serve
```
